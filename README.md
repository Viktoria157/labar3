# 📌Лабораторна робота №3
## Короткий опис задачі:
Необхідно створити програму мовою C++, яка працює з квадратною матрицею розміру 3×3.
Програма повинна:

ввести елементи матриці з клавіатури;

знайти добуток від’ємних елементів у парних рядках матриці;

якщо таких елементів немає — вивести відповідне повідомлення;

відсортувати елементи кожного рядка матриці методом бульбашки;

вивести відсортовану матрицю.

Опис алгоритму розв’язання

Алгоритм роботи програми складається з таких кроків:

Початок роботи програми.

Оголошення змінних:

матриця a[3][3];

змінна dob для обчислення добутку;

логічна змінна found для перевірки наявності від’ємних елементів.

Введення елементів матриці користувачем.

Перевірка парних рядків матриці.

Пошук від’ємних елементів у цих рядках.

Якщо елемент від’ємний — він множиться на змінну dob.

Якщо знайдено хоча б один від’ємний елемент — виводиться добуток.

Якщо від’ємних елементів немає — виводиться повідомлення.

Виконується сортування кожного рядка матриці методом бульбашки.

Виводиться відсортована матриця.

Завершення роботи програми.

Пояснення основних етапів роботи програми
1. Введення матриці

Користувач вводить 9 чисел, які формують квадратну матрицю 3×3.

Для цього використовуються вкладені цикли for, які проходять по рядках та стовпцях матриці.

2. Пошук добутку від’ємних елементів у парних рядках

Програма перевіряє кожний рядок матриці.

Рядок вважається парним, якщо виконується умова:

(i + 1) % 2 == 0

У цих рядках перевіряються всі елементи.
Якщо елемент менший за нуль, він множиться на змінну dob.

Логічна змінна found показує, чи були знайдені від’ємні елементи.

3. Виведення результату

якщо від’ємні елементи знайдені — виводиться їх добуток;

якщо їх немає — виводиться повідомлення:

No negative elements in even rows
4. Сортування матриці

Після обчислень виконується сортування елементів кожного рядка методом бульбашки.

Алгоритм бульбашки працює так:

порівнюються сусідні елементи;

якщо перший більший за другий — вони міняються місцями;

процес повторюється кілька разів.

5. Виведення відсортованої матриці

Після завершення сортування програма виводить оновлену матрицю у вигляді таблиці 3×3.
